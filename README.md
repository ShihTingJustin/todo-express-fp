# todo-express-fp
A todo list api server built with Express.js.

## Setup
1. clone repository
```
git clone https://github.com/ShihTingJustin/todo-express-fp
```
2. go to project directory
```
cd todo-express-fp
```
3. install dependencies
```
yarn
```
4. serving the project
```
yarn dev
```
5. use it on
```
http://127.0.0.1:3001/api-docs
```

## Stack
* TypeScript
* Express
* Node.js
* swagger
* Jest
* fp-ts
